# Claude Session Usage — расширение Chrome

Показывает прогресс 5-часовой сессии прямо на странице **claude.ai**.

## Как установить (без публикации в Chrome Web Store)

1. Скачай и распакуй папку `claude-usage-extension`
2. Открой Chrome → адресная строка → `chrome://extensions`
3. Включи **«Режим разработчика»** (правый верхний угол)
4. Нажми **«Загрузить распакованное расширение»**
5. Выбери папку `claude-usage-extension`
6. Открой `https://claude.ai` — виджет появится в правом нижнем углу

## Что показывает

| Элемент | Описание |
|---------|----------|
| **% использования** | Сколько лимита 5-часовой сессии потрачено |
| **Полоска** | Зелёная → жёлтая → красная по мере заполнения |
| **Осталось** | Сколько времени до сброса сессии |
| **Сброс в** | Точное время сброса |

## Управление виджетом

- **Перетаскивание** — зажми заголовок и тяни куда нужно
- **↻** — обновить данные вручную
- **×** — свернуть / развернуть виджет

## Автообновление

Данные обновляются автоматически каждую минуту.
